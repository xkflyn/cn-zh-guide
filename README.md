# cn-zh-guide

本仓库用于归档和发布多个独立 HTML 页面，内容以中文为主，涵盖不同主题与用途。

## 项目简介

这是一个静态页面集合仓库，主要用于存放和展示独立的 HTML 文档。仓库内页面彼此独立，不依赖于特定域名或网站，适合作为信息存档、学习参考或轻量级工具页面使用。

## 目录说明

```
cn-zh-guide/
├── pages/          # 存放所有独立 HTML 页面
├── assets/         # 公共资源文件（CSS、图片、字体等）
├── index.html      # 仓库入口页面（可选）
└── README.md       # 本文件
```

- **pages/**：每个 HTML 页面为一个独立文件，命名清晰，便于直接访问。
- **assets/**：存放页面所需的公共静态资源，避免重复引用。

## 页面归档说明

所有页面均为静态 HTML，不包含后端逻辑或动态内容。页面内容可能涉及知识整理、操作指引、工具界面等，具体以页面本身描述为准。归档时不针对任何特定网站或服务，仅作为独立文档呈现。

## 维护说明

- 欢迎提交 Issue 或 Pull Request 来完善内容、修复错误或新增页面。
- 新增页面请放置在 `pages/` 目录下，并保证页面内资源引用路径正确。
- 请勿在页面中添加任何营销、推广或诱导点击内容。
- 本仓库不承诺持续更新，但会尽量保持内容可用与准确。
